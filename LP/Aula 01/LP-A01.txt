[LP-A01] Qual das alternativas abaixo representa corretamente a sintaxe de uma variável em python?

a) (nome = "João"
b) 1idade = 12
c) True = True
d) cEp = '41.340-100'
e) falso = false


Explicação
d) cEp = '41.340-100': Nesta sintaxe, uma variável chamada "cEp" está sendo declarada e atribuída com o valor '41.340-100'. Em Python, o nome de uma variável deve começar com uma letra (maiúscula ou minúscula) seguido por letras, dígitos ou sublinhados. É importante observar que o Python é sensível a maiúsculas e minúsculas, então "cEp" é diferente de "cep".

a) (nome = "João": Os parênteses não são necessários e podem causar um erro de sintaxe.

b) 1idade = 12: Começar o nome de uma variável com um número é inválido.

c) True = True: "True" é uma palavra reservada em Python e não pode ser usada como o nome de uma variável.

e) falso = false: "false" é uma palavra reservada em Python, mas deve ser escrita com a primeira letra minúscula ("False") para representar o valor booleano falso.


[LP-A01] Qual das opções abaixo representa o comportamento do sinal "=" em python:

a) igualdade
b) atribuição
c) comparação
d) diferença
e) negação

Explicação
b) atribuição: Em Python, o sinal "=" é utilizado para atribuir um valor a uma variável. Ele indica que o valor à direita do sinal será atribuído à variável à esquerda.
