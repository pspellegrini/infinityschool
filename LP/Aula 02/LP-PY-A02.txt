[LP-PY-A02] Levando em consideração o código abaixo:
x = 4
if x == 4:
    print("x é igual a 4")
else:
    print("x é diferente de 4")

Marque a alternativa CORRETA.

a) O trecho if x == 4: poderia ser substituído por if 4 = x: e ainda estaria correto.
b) Caso o else fosse retirado do código, o programa somaria 1 + 1.
c) O resultado imprimido por print("x é == a 4") é: "x é igual a 4"
d) A atribuição da variável x = 4 está incorreta, o correto é x == 4
e) O trecho else: print("x é diferente de 4") poderia ser substituído por: elif x != 4: print("x é diferente de 4")

Gabarito:
e) O trecho else: print("x é diferente de 4") poderia ser substituído por: elif x != 4: print("x é diferente de 4")
