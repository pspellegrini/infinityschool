ATIVIDADE PRÁTICA

  1. Crie uma Lista Não Ordenada:
    • Use a tag "< ul >" para criar uma lista não ordenada.
    • Adicione pelo menos três itens à lista usando a tag "< li >".
  2. Crie uma Lista Ordenada:
    • Utilize a tag "< ol >" para criar uma lista ordenada.
    • Insira pelo menos três itens na lista, também com a tag "< li >".
  3. Dentro de um item de qualquer lista acima, crie uma sublista aninhada. Pode ser uma lista ordenada dentro de uma não ordenada, ou vice-versa.

    4. Por exemplo, se você tem um item "Receitas" na lista não ordenada, crie uma sublista ordenada com "Passo 1", "Passo 2", etc., dentro dele.
